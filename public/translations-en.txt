faq.label = FAQ
faq.title = Common questions
faq.q1 = How accurate are the solar forecasts?
faq.a1 = Volcast uses a full physical irradiance model combined with satellite weather data and your specific system parameters. After calibration with real production data, accuracy typically reaches 85–95% on clear and partly cloudy days. Heavy cloud cover is harder to predict for any system, but Volcast still outperforms generic weather apps.
faq.q2 = Do I need a smart inverter or specific hardware?
faq.a2 = No. Volcast works with any solar installation — no inverter cloud login, no API keys, no hardware connection. You just enter your panel parameters: azimuth, tilt, peak power, and optionally multi-string setup, bifacial factor, and horizon shading profile.
faq.q3 = Is Volcast free?
faq.a3 = Volcast offers a free tier with 2-day forecasts. Premium unlocks the full 7-day forecast, auto-calibration, REST API for Home Assistant, production history, and more — as a one-time purchase, no subscription.
faq.q4 = Which platforms are supported?
faq.a4 = Volcast is available now on Android (Google Play). iOS is coming soon — join the waitlist above to be notified. Home screen widgets work on both Android (Glance) and iOS (WidgetKit).
faq.q5 = How does auto-calibration work?
faq.a5 = You log your actual daily production (just one number). After 5+ data points, the Kalman filter compares your real output against the physical model to fine-tune parameters specific to your installation — accounting for shading, panel degradation, and local factors.
faq.q6 = How does the Home Assistant integration work?
faq.a6 = Volcast provides a REST API endpoint. You add it as a sensor in your configuration.yaml with your API key (generated in the app, Premium feature). Home Assistant polls the endpoint hourly and gets daily forecasts, hourly breakdowns, and 5-minute detailed data for today and tomorrow.
